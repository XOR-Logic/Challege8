WEB DEVELOPMENT

Mempelajari bahasa pemrograman yang dipakai untuk membuat website.

HTML Dasar

Meurpakan dasar dari semua halaman web di internet. Jika anda ingin mempelajari cara membuat website, maka Tutorial Belajar HTML Dasar Untuk Pemula adalah langkah pertama anda

CSS Dasar

CSS adalah kumpulan kode program yang digunakan untuk mendesain atau mempercantik tampilan halaman HTML. Dengan CSS kita bisa mengubah desain dari text, warna, gambar dan latar belakang dari (hampir) semua kode tag HTML.

Javascript Dasar

Javascript adalah bahasa pemrograman yang awalnya dirancang untuk berjalan di atas browser. Javascript juga dapat digunakan pada sisi Server, Game, IoT, Desktop, dsb.

JQuery Dasar

JQuery adalah sebuah library framework javascript, JQuery merupakan pengembangan dari javascript yang sengaja di buat untuk memudahkan kita untuk mendevelop sebuah website dan ingin membuat efek-efek dan keperluan penggunaan javascript lainnya di dalam pengembangan sebuah website.

Ajax Dasar

AJAX adalah sebuah singkatan dari Asynchronous Javascript and XML dan mengacu pada sekumpulan teknis pengembangan web (web development) yang memungkinkan aplikasi web untuk bekerja secara asynchronous (tidak langsung) â€“ memproses setiap request (permintaan) yang datang ke server di sisi background.

Boostrap Dasar

Bootstrap adalah sebuah library framework CSS yang di buat khusus untuk bagian pengembangan front-end website. bootstrap merupakan salah satu framework HTML, CSS dan Javascript
